CSS Positioning

Static is the default setting and renders the elements in order of the document flow.

Relative means that the element is positioned relative to its normal position, it falls naturally but you can add top, right, left, bottom properties.

Absolute lets you target the position you want within a relative element.

Fixed is a fixed position within the browser window.

Initial sets the property to its default value.

Inherit: the element inherits the value of its parent element.

CSS Positioning Types

  • Static
  • Relative
  • Absolute
  • Fixed
  • Initial
  • Inherit
